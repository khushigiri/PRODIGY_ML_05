# Food Calorie Estimator

A simple and interactive **AI-powered web application** that detects food items from images and estimates their calorie content. This project helps users track their dietary intake and make smarter food choices.

---

## Features

* Upload food images for instant analysis
* Food classification using deep learning
* Calorie estimation for detected food
* Fast and user-friendly interface with Streamlit

---

## How It Works

1. User uploads a food image
2. The trained model predicts the food item
3. The app maps the prediction to its calorie value
4. Results are displayed instantly

---

## Tech Stack

* **TensorFlow (MobileNetV2)** – for image classification
* **Python** – backend logic
* **Streamlit** – web interface

---

## Limitations

* The model can only predict food categories it was trained on
* Calorie values are approximate (based on predefined mapping)
* Accuracy depends on dataset quality
